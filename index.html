üìñ Quick Start Guide

üîí How to Encrypt Files

  1. Select Files: Drag & drop or click to select files
  2. Set Passphrase: Use a strong, unique passphrase
  3. Choose Iterations: 250k recommended for balance
  4. Select Folder: Choose save location (optional)
  5. Encrypt: Click "Encrypt & Save" button

üîì How to Decrypt Files

  1. Select .enc Files: Choose encrypted files
  2. Enter Passphrase: Use exact encryption passphrase
  3. Match Iterations: Use same iterations as encryption
  4. Select Folder: Choose save location (optional)
  5. Decrypt: Click "Decrypt & Save" button

‚ùì Frequently Asked Questions

What if I forget my passphrase?

Files cannot be recovered without the exact passphrase. We don't store keys or passphrases. Always keep backups of your passphrases in a secure location.

Is this tool really secure?

Yes! All encryption/decryption happens locally in your browser using Web Crypto API. No data is transmitted to servers. Your files never leave your device.

Which browsers are supported?

Chrome, Edge, Firefox, and Safari (latest versions) with Web Crypto API support. For best performance, use updated browsers.

Can I encrypt large files?

Yes, but performance depends on your device. For files over 1GB, consider splitting them. Browser memory limits may apply for very large files.
